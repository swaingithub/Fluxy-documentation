## üå™Ô∏è Smart Navigation (`Fx.to`)

Fluxy includes a global routing proxy that removes the need for `Navigator.of(context)` everywhere.

```dart
// Standard
Navigator.pushNamed(context, '/settings');

// Fluxy (Context-less)
Fx.to('/settings');
Fx.back();
```

## üöß Common "Gotchas" & Solutions

| Issue | Standard Flutter Pain | Fluxy Solution |
| :--- | :--- | :--- |
| **Nested Nav Crashing** | `Hero` conflicts and routing loops. | **Fluxy Router**: A stabilized, single-instance routing engine. |
| **Drawer Overlays** | Manual `GlobalKey<ScaffoldState>`. | Integrated Proxy: `Fx.toggleDrawer()` works from anywhere. |
| **Bottom Bar Overlap** | Floating Action Button collisions. | `Fx.page` automatically calculates safe areas and FAB floating. |
