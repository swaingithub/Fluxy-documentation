# Fluxy CLI

The Fluxy CLI is a powerful tool to automate your workflow, manage projects, and run development tasks.

## Installation

Install globally via Dart:

```bash
dart pub global activate fluxy
```

## Core Commands

### `fluxy init`

Initializes a new Fluxy project structure.

```bash
fluxy init
```

### `fluxy doctor`

Checks your environment for potential issues.

```bash
fluxy doctor
```

### `fluxy run`

Runs your application in debug mode. This is a wrapper around `flutter run` with enhancements.

```bash
fluxy run
```

### `fluxy build`

Builds your application for production.

```bash
fluxy build apk
fluxy build ios
fluxy build web
```

## Generators

### `fluxy gen:page <Name>`

Generates a new View, Controller, and Route entry.

```bash
fluxy gen:page Home
```

### `fluxy gen:controller <Name>`

Generates a standalone reactive controller.

```bash
fluxy gen:controller Auth
```
