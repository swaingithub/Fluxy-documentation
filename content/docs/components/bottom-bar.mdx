# FxBottomBar

`FxBottomBar` is a navigation component designed for primary app shells. It features physics-based selection tracking and smooth transitions.

## Usage

```dart
FxBottomBar(
  selectedIndex: currentTab, // Signal<int>
  items: [
    FxBottomBarItem(icon: Icons.home, label: "Home"),
    FxBottomBarItem(icon: Icons.search, label: "Search"),
    FxBottomBarItem(icon: Icons.person, label: "Profile"),
  ],
  onTap: (index) => currentTab.value = index,
)
```

## API Reference

| Name | Type | Default | Description |
|------|------|---------|-------------|
| `selectedIndex` | `Signal<int>` | required | The active tab index. |
| `items` | `List<FxBottomBarItem>` | required | List of navigation items. |
| `onTap` | `ValueChanged<int>?` | `null` | Optional callback when tab is selected. |
| `style` | `FxStyle?` | `null` | Custom styling for the bar container. |

## The Master Navigation Implementation

A modern, glassmorphic bottom navigation bar with integrated notification badges.

```dart
class AppShellNavigation extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    final activeTab = flux(0);
    final notificationCount = flux(3);

    return Fx.bottomBar(
      selectedIndex: activeTab,
      onTap: (i) => activeTab.value = i,
      style: FxStyle(
        bg: Colors.white.withOpacity(0.8),
        blur: 20, // Glassmorphic effect
        border: Border(top: BorderSide(color: Colors.slate200, width: 1)),
        p: 12,
      ),
      items: [
        FxBottomBarItem(icon: Icons.grid_view_rounded, label: "Home"),
        FxBottomBarItem(
          icon: Icons.chat_bubble_outline_rounded, 
          label: "Messages",
          // Integrated badge for navigation items
          badge: Fx.badge(label: notificationCount.value.toString(), color: Colors.red),
        ),
        FxBottomBarItem(icon: Icons.account_circle_outlined, label: "Profile"),
      ],
    );
  }
}
```

Fluxy's navigation system utilizes high-performance physics-based animations for selection indicators, providing a premium "liquid" feel that is characteristic of high-end industrial applications.
