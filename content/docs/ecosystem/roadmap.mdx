# Roadmap & Status

Fluxy is evolving rapidly. Here is the current status and future plans as of **v0.2.5 (The Managed Platform Era)**.

## Current Status (v0.2.5)
- [x] **Managed Platform Layer**: Production Ready (`Fx.platform`)
- [x] **Auto-Registration**: Production Ready (`Fluxy.autoRegister`)
- [x] **Signals & State Engine**: Production Ready
- [x] **Scoped DI (FxScope)**: Production Ready
- [x] **Fluxy Inspector**: Production Ready (Glassmorphic HUD)
- [x] **Official Modules (Auth, Storage, etc)**: Production Ready
- [x] **Managed Networking (FluxyHttp)**: Production Ready
- [x] **Reactive Routing**: Production Ready
- [x] **Stability Kernel**: Production Ready (Auto-repair system)

## Future Plans (2026)
- [ ] **VS Code Extension**: Snippets and interactive DSL preview.
- [ ] **Animation DSL Integration**: Expressive, chainable animation engine.
- [ ] **Server-Side Rendering (SSR)**: Experimental support for Flutter Web.

## Stable V1.0 Release
- [ ] Performance Benchmarks
- [ ] Full Documentation Coverage
- [ ] Stable API Freeze
