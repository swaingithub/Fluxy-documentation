# Examples Gallery

Explore how Fluxy can be used to build beautiful, reactive Flutter applications.

## 1. Minimal Counter

The classic counter app, optimized with atomic reactivity.

```dart
final count = Signal(0);

Fx.button("Count: ${count.value}")
  .onPressed(() => count.value++)
  .render();
```

## 2. Dynamic List

A list that updates efficiently when items are added or removed.

```dart
final items = Signal<List<String>>([]);

Fx.column()
  .children([
    Fx.button("Add Item").onPressed(() => items.value = [...items.value, "New Item"]),
    ...items.value.map((item) => Fx.text(item).render()),
  ])
  .render();
```

## 3. Form Validation

Reactive form validation without the boilerplate of `Form` and `TextFormField`.

```dart
final email = Signal("");
final isValid = Computed(() => email.value.contains("@"));

Fx.column()
  .children([
    Fx.textField().onChanged((v) => email.value = v),
    if (!isValid.value) Fx.text("Invalid Email").red().render(),
  ])
  .render();
```
