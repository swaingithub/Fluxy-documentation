# Examples & Recipes

Explore how to build modern apps using the Fluxy v0.1.1 feature set.

## Counter App (Atomic)

The classic counter example demonstrating fine-grained reactivity with the `Fx` builder.

```dart
final count = flux(0);

Widget build(BuildContext context) {
  return Fx.column()
    .center()
    .gap(16)
    .children([
      // Only this part rebuilds when count changes
      Fx(() => Fx.text("Count: ${count.value}").h1().render()),
      
      Fx.box()
        .p(16)
        .bg(FxTokens.primary)
        .rounded(12)
        .onPressed(() => count.value++)
        .child(Fx.text("Increment").white().render())
        .render(),
    ])
    .render();
}
```

## User Profile (Premium Widgets)

Using `FxAvatar` and `FxBadge` with conditional styling.

```dart
Widget UserProfile() {
  return Fx.row()
    .p(16)
    .bg(Colors.grey.withValues(alpha: 0.1))
    .rounded(FxTokens.radius.lg)
    .children([
      FxBadge(
        label: "Online",
        child: FxAvatar(
          image: "https://i.pravatar.cc/150",
          size: FxAvatarSize.md,
        ),
      ),
      Fx.column()
        .px(16)
        .crossStart()
        .children([
          Fx.text("Jane Doe").bold().render(),
          Fx.text("Software Engineer").size(12).render(),
        ])
        .render(),
    ])
    .render();
}
```

## Form with Two-Way Binding

Native two-way binding using `FxTextField` and `FxDropdown`.

```dart
final username = flux("");
final role = flux("Developer");

Widget RegistrationForm() {
  return Fx.column()
    .p(24)
    .gap(FxTokens.space.md)
    .children([
      FxTextField(
        label: "Username",
        value: username, // Auto-binds to signal
      ),
      FxDropdown<String>(
        label: "Select Role",
        value: role,
        options: ["Developer", "Designer", "Manager"],
      ),
      Fx.button("Submit")
        .px(32)
        .py(12)
        .bg(FxTokens.primary)
        .onPressed(() {
          print("Registering ${username.value} as ${role.value}");
        })
        .render(),
    ])
    .render();
}
```

## Hover Effects

Adding interactive states using the Atomic DSL.

```dart
Fx.box()
  .square(150)
  .bg(Colors.blue)
  .rounded(FxTokens.radius.md)
  .onHover((s) => s.scale(1.1).bg(Colors.blueAccent))
  .onPressed((s) => s.op(0.7))
  .center()
  .child(Fx.text("Interactive").white().render())
  .render();
```
