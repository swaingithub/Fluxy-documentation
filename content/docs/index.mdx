# The Full-Stack SDK for Flutter

Fluxy is a production-grade **Managed Application Platform** that replaces the "glue code" in your Flutter apps. It unifies specific tooling for State, DI, Networking, and UI into one cohesive engine with a **modular architecture** (v1.0.0).

- **Separate Packages**: Add only the features you need
- **Professional Logging**: Industrial-grade logging with semantic tags
- **Experimental Guardrails**: Clear warnings for advanced features
- **Migration Support**: Seamless upgrade from v0.2.6

## üèóÔ∏è Modular Architecture (v1.0.0)

Fluxy v1.0.0 introduces a revolutionary modular approach. Install only what you need:

### Core Framework (172KB)
```yaml
dependencies:
  fluxy: ^1.0.0
```

### Platform Modules (Add as needed)
```yaml
dependencies:
  fluxy_auth: ^1.0.0         # Authentication & biometrics
  fluxy_camera: ^1.0.0       # Camera functionality
  fluxy_notifications: ^1.0.0 # Push notifications
  fluxy_storage: ^1.0.0      # Data persistence
  # ... and 8 more modules
```

### Benefits
- **Faster Installation**: 172KB core vs 15MB+ monolithic
- **Flexible Dependencies**: Add only features you use
- **Independent Versioning**: Each module can evolve separately
- **Clear Architecture**: Separated concerns for better maintainability

## Architecture that Scales

Fluxy enforces a clean, consistent structure for your team.

- **Controllers**: Logic lives outside the UI.
- **Services**: Global singletons for data.
- **Repositories**: Standardized data access.

```dart
class UserController extends FluxController {
  final user = flux<User?>(null);
  
  void login() async {
    user.value = await api.login();
  }
}
```
