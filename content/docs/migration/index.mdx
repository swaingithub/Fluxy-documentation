# Migration Guides

This section contains migration guides for different versions of Fluxy to help you upgrade your applications smoothly.

## Available Guides

- [v0.1.8 Migration Guide](./docs/migration/migration-0-1-8) - Latest migration guide covering the new Fluxy State System with branded keywords, advanced performance features, and offline-first architecture.
- [v0.1.7 Migration Guide](./docs/migration/migration-0-1-7) - Migration guide for v0.1.7 features and improvements.
- [v0.1.6 Migration Guide](./docs/migration/migration-0-1-6) - Migration guide for v0.1.6 features and improvements.

---

## General Migration Tips

1. **Always backup your project** before upgrading
2. **Read the specific migration guide** for your target version
3. **Test thoroughly** after upgrading
4. **Check deprecation warnings** for future compatibility
5. **Update dependencies** to compatible versions
