This section contains migration guides for different versions of Fluxy to help you upgrade your applications smoothly.

- [v1.0.0 Migration Guide](./migration-1-0-0) - Modular architecture and multi-package ecosystem

---

## General Migration Tips

1. **Always backup your project** before upgrading
2. **Read the specific migration guide** for your target version
3. **Test thoroughly** after upgrading
4. **Check deprecation warnings** for future compatibility
5. **Update dependencies** to compatible versions
