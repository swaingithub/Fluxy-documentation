# Migration to v0.1.11

Fluxy v0.1.11 and v0.1.10 introduce stability improvements, architectural automation, and a slight breaking change in how font sizes are accessed in the Styling DSL.

## Breaking Changes (v0.1.10)

### Font Proxy: Getters to Methods

To ensure stable chaining and prevent runtime invocation errors, font size proxies have been reverted from **getters** to **methods**. You must now invoke font sizes with parentheses.

**Before (v0.1.9):**
```dart
Fx.text("Hello")
  .font.lg.bold // 'lg' was a getter
```

**After (v0.1.10+):**
```dart
Fx.text("Hello")
  .font.lg().bold // 'lg' is now a method
```

**Impacted Tokens:**
- `xs()`, `sm()`, `md()`, `lg()`, `xl()`, `xl2()`, `xl3()`, `xl4()`, `xl5()`, `xl6()`

---

## New Features

### v0.1.11: Automatic Route DI

The biggest improvement in v0.1.11 is the automatic registration and disposal of controllers defined within `FxRoute`. You no longer need to manually call `FluxyDI.put()` or `FluxyDI.delete()` for these controllers.

**Before:**
```dart
FxRoute(
  path: '/profile',
  builder: (context) => ProfilePage(),
  controller: (context) {
    final c = ProfileController();
    FluxyDI.put(c); // Manual registration
    return c;
  },
)
```

**After:**
```dart
FxRoute(
  path: '/profile',
  builder: (context) => ProfilePage(),
  controller: (context) => ProfileController(), // Auto-registered by framework
)
```

### v0.1.10: Expanded Typography

Three new font size tokens have been added to the design system:
- `xl4()`: Extra large 4
- `xl5()`: Extra large 5
- `xl6()`: Extra large 6

New text modifiers:
- `.italic()`: Apply italic style.
- `.letterSpacing(double)`: Semantically adjust spacing between letters.

---

## Stability Fixes

- **OTA Engine**: Fixed an issue where the core OTA/SDUI engine files were accidentally excluded from the package distribution.
- **Blueprint Polish**: Fixed type-mismatch errors in the `feed` blueprint.
- **DI Engine**: Improved internal framework orchestration with `putByRuntimeType`.

## Recommended Upgrade Path

1. Update your `pubspec.yaml` to `fluxy: ^0.1.11`.
2. Run `flutter pub get`.
3. Global search for `.font.` and add `()` to all size tokens (e.g., `.font.md` -> `.font.md()`).
4. (Optional) Simplify your `FxRoute` definitions by removing manual `FluxyDI` calls.
