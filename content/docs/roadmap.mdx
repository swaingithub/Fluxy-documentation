# Roadmap

Fluxy is evolving rapidly. Here's what we have planned for the near future.

## Q1 2024: Stability & Core (Current)
- [x] Alpha Release
- [x] Core Reactive Engine
- [x] Basic UI DSL (Text, Box, Rows, Columns)
- [ ] Comprehensive Unit Tests
- [ ] Documentation Website (You are here!)

## Q2 2024: Ecosystem & Tooling
- [ ] Fluxy CLI for scaffolding
- [ ] VS Code Extension (Snippets & DevTools)
- [ ] Responsive Layout Engine Refinement
- [ ] Animation DSL Integration

## Q3 2024: Advanced Features
- [ ] Global Navigation System
- [ ] Direct Form Validation DSL
- [ ] Integrated Persisted State (Local Storage)
- [ ] Server-Side Rendering (SSR) Experimental support for Flutter Web

## Q4 2024: V1.0 Stable
- [ ] Beta Release
- [ ] Performance Benchmarks
- [ ] Community Plugins support
- [ ] Stable V1.0 Release
